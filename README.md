# c-four
## About
c-four(Clean Commit Co-Creator) is the useful console tool to co-create the clean commit with you.
## Install
```
$ pip install wheel
$ pip install git+https://github.com/tklab-group/c-four
```
## Usage
```
$ c-four
```
